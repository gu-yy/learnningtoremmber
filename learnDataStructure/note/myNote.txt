2、
	传递引用传递的只是实参的一个别名，相当于在被调函数中调用的是外部函数中的实参的一个快捷方式，在被调函数的入口处，也会对传递进来的形参进行复制，但是这个复制
	的是一个引用，造成的结果是对外界函数的实参的快捷方式多了一份，所以在被调函数内对形参进行操作的时候实质上是对外界函数的实参的引用也就是快捷方式进行操作，而
	这个对引用（快捷方式）的操作实质上就是对外部函数实参的操作，所以传递引用也能够改变外部参数的值，因此与值传递的结果不同
		地址传递：
		f(int *a);
		int main(){
			f(&a);//这里的&不是引用的意思，而是对参数去地址的意思
		}
		引用传递
		f(int &a);
		int main(){
			f(a);//这里的&是引用的意思，传递引用的调用方式和值传递的调用方式相同
		}

	当A是对象时，调用A的属性B：用点号.：A.B
	当A是指针时，调用A的属性B：用箭头->：A->B
